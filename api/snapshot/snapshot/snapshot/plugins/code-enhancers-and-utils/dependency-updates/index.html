dependency-updates

A custom plugin for to find the latest versions for all dependencies which uses ben-manes under the hood.

Download from MavenCentral

dependencies {
    implementation("com.javiersc.gradle-plugins:dependency-updates:$version")
}

Add to gradle.properties the property dependencyDiscoveryLimit to define the limit for the dependencies.

Options:

  • SNAPSHOT

  • dev

  • eap

  • alpha

  • beta

  • M (or milestone)

  • rc

  • stable

Samples

dependencyDiscoveryLimit=stable
